My dotfiles
===========

Just my dotfiles, for shells, iTerm2 and vim. These are designed for use
on OS X but should (particularly the shells and vim stuff) work
elsewhere.

Many of the config files started off when I was using Debian GNU/Linux
as my main OS. I got the structure of this repo from [Jack
Franklin](https://github.com/jackfranklin/dotfiles) and many others.
