# My dotfiles

Just my dotfiles and mostly-automated steps for getting core (to me) software installed.

The `packages.csv` file contains the package names for that software across platforms. When it's updated, running `packages.pl` will update the `Makefile` accordingly.

## Stow

[GNU Stow](https://www.gnu.org/software/stow/) is used as per [Using GNU Stow to manage your dotfiles](https://brandon.invergo.net/news/2012-05-26-using-gnu-stow-to-manage-your-dotfiles.html) by Brandon Invergo.
